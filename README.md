# NAHUI3 Documentation Site

Статический сайт документации для проекта NAHUI3, размещённый на GitHub Pages.

## Структура

- `index.html` - главная страница с обзором всех репозиториев проекта
- `styles.css` - стили для сайта

## Развёртывание

Сайт автоматически развёртывается через GitHub Pages при пуше в репозиторий `nahui3.github.io`.

## Локальный просмотр

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

## Репозитории проекта

- [backend](https://github.com/nahui3/backend) - Go REST API
- [admin](https://github.com/nahui3/admin) - Django админ-панель
- [android](https://github.com/nahui3/android) - Android библиотека
- [ios](https://github.com/nahui3/ios) - iOS библиотека
- [web](https://github.com/nahui3/web) - Web библиотека
- [demo](https://github.com/nahui3/demo) - Демо проекты
